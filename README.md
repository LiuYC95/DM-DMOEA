# DM-DMOEA
This is the matlab code for DM-DMOEA. Y. Liu, L. Tang, J. Ding, Q. Chen, K. Liu and J. Liu, "A Dual Mutation-Based Evolutionary Algorithm for Dynamic Multiobjective Optimization With Undetectable Changes," in IEEE Transactions on Evolutionary Computation, vol. 29, no. 4, pp. 1199-1214, Aug. 2025, doi: 10.1109/TEVC.2024.3424393.
